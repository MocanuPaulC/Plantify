About Plantify

Plantify is the first digital watering system that has a built-in reservoir.
It was created to make it easier to take care of plants by notifying the user that they need to water
their plant or that they need to refill the reservoir. By building an easy-to-understand platform,
makes creating an account and installing the device an easy task.

The watering system was created using Arduino components and C programming,
while the self-watering pot is made using 3D printing and connected via Wi-Fi to make Plantify
